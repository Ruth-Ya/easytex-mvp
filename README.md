# EasyTex — MVP (Vite + React + Tailwind)

Prototype fonctionnel pour tests marché (WhatsApp devis + formulaire fournisseurs).

## Démarrer en local
```bash
npm install
npm run dev
```

## Build de production
```bash
npm run build
npm run preview
```

## Déploiement Vercel
- Framework: **Vite**
- Build command: `npm run build`
- Output dir: `dist`
